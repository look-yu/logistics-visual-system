# 订单价格计算 API 说明文档

## 📋 功能概述

新增订单功能已升级，现在支持完整的货物信息录入和自动价格计算。

## 🎯 新增功能

### 1. 货物信息录入
- **货物类型**：普通货物、生鲜食品、电子产品、危险品、贵重物品、大件货物
- **重量**：支持0.1-10000公斤，精确到小数点后2位
- **体积**：支持0.01-1000立方米，精确到小数点后2位

### 2. 价格自动计算
- **基础运费**：用户手动输入
- **重量附加费**：重量 × 重量单价（默认2元/公斤）
- **体积附加费**：体积 × 体积单价（默认50元/立方米）
- **货物类型附加费**：基础运费 × (类型倍数 - 1)
- **总金额**：基础运费 + 重量附加费 + 体积附加费 + 货物类型附加费

### 3. 货物类型价格倍数
| 货物类型 | 价格倍数 | 说明 |
|---------|---------|------|
| 普通货物 | 1.0 | 无额外费用 |
| 生鲜食品 | 1.5 | +50% |
| 电子产品 | 1.3 | +30% |
| 危险品 | 2.0 | +100% |
| 贵重物品 | 2.5 | +150% |
| 大件货物 | 1.8 | +80% |

## 📡 API 接口说明

### 1. 获取价格配置

**接口地址：** `GET http://localhost:5001/api/price/config`

**请求参数：** 无

**响应示例：**
```json
{
  "code": 200,
  "data": {
    "base_price": 100.00,
    "weight_price": 2.00,
    "volume_price": 50.00,
    "goods_type_multiplier": {
      "普通货物": 1.0,
      "生鲜食品": 1.5,
      "电子产品": 1.3,
      "危险品": 2.0,
      "贵重物品": 2.5,
      "大件货物": 1.8
    }
  },
  "msg": "获取价格配置成功"
}
```

---

### 2. 更新价格配置

**接口地址：** `PUT http://localhost:5001/api/price/config`

**请求参数：**
```json
{
  "base_price": 100.00,
  "weight_price": 2.00,
  "volume_price": 50.00,
  "goods_type_multiplier": {
    "普通货物": 1.0,
    "生鲜食品": 1.5,
    "电子产品": 1.3,
    "危险品": 2.0,
    "贵重物品": 2.5,
    "大件货物": 1.8
  }
}
```

**响应示例：**
```json
{
  "code": 200,
  "msg": "价格配置更新成功"
}
```

---

### 3. 计算订单价格

**接口地址：** `POST http://localhost:5001/api/price/calculate`

**请求参数：**
```json
{
  "goods_type": "生鲜食品",
  "weight": 10.5,
  "volume": 2.3,
  "base_price": 100.0
}
```

**响应示例：**
```json
{
  "code": 200,
  "data": {
    "base_price": "100.00",
    "weight_fee": "21.00",
    "volume_fee": "115.00",
    "goods_type_fee": "50.00",
    "total_amount": "286.00",
    "breakdown": {
      "weight_price": 2.0,
      "volume_price": 50.0,
      "multiplier": 1.5
    }
  },
  "msg": "价格计算成功"
}
```

**计算说明：**
- 基础运费：100.00 元
- 重量附加费：10.5 × 2.0 = 21.00 元
- 体积附加费：2.3 × 50.0 = 115.00 元
- 货物类型附加费：100.0 × (1.5 - 1.0) = 50.00 元
- **总金额：100.00 + 21.00 + 115.00 + 50.00 = 286.00 元**

---

## 🎨 前端使用说明

### 1. 打开新增订单对话框
点击"订单管理"页面中的"新增订单"按钮。

### 2. 填写基本信息
- **客户名称**：必填
- **发货地址**：必填
- **收货地址**：必填

### 3. 填写货物信息
- **货物类型**：从下拉菜单选择
- **重量**：输入重量（公斤）
- **体积**：输入体积（立方米）

### 4. 设置价格信息
- **基础运费**：输入基础运费（元）
- 系统会自动计算：
  - 重量附加费
  - 体积附加费
  - 货物类型附加费
  - 总金额

### 5. 提交订单
点击"确定"按钮提交订单。

- 货物类型附加费：100.0 × (2.0 - 1.0) = 100.00 元

## 🔧 数据库表结构

### price_config 表

| 字段名 | 类型 | 说明 | 默认值 |
|-------|------|------|-------|
| id | int | 主键 | 1 |
| base_price | decimal(10,2) | 基础运费 | 100.00 |
| weight_price | decimal(10,2) | 重量单价（元/公斤） | 2.00 |
| volume_price | decimal(10,2) | 体积单价（元/立方米） | 50.00 |
| goods_type_multiplier | json | 货物类型价格倍数 | JSON对象 |
| create_time | timestamp | 创建时间 | CURRENT_TIMESTAMP |
| update_time | timestamp | 更新时间 | CURRENT_TIMESTAMP |

---

## 📝 注意事项

1. **价格配置**：价格配置存储在数据库中，可以通过API动态修改
2. **表单验证**：所有字段都有必填验证，确保数据完整性
3. **实时计算**：修改任何字段都会实时更新价格计算结果
4. **精度处理**：所有金额都保留2位小数
5. **默认值**：新增订单时会自动填充默认值，方便快速录入
